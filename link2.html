상대경로

절대적 웹 주소가 아닌 현재 작업중인 파일을 기준으로 해당 폴더와 같은 폴더 또는 하위 폴더 또는
상위 폴더로 원하는 파일의 위치를 상대적으로 지정하는 경로.

절대경로 웹주소 작성시 시작 -> http

상대경로 작성 시 시작 -> ./ | '현재 위치 부터 시작해라' 라는 뜻

1. 현재 작업중인 파일의 위치를 확인한다.

2. 위 1 기준 연결하고 싶은 파일의 위치를 확인한다.

3. ./로 경로를 시작하며 원하는 상대 경로를 완성한다.

시작 파일 보기(같은위치 기준)
인형찾기(하위경로 기준으로 확인)
노트북 찾기
휴지 찾기
지우개 찾기

상위 경로

현재 작업파일 기준으로 연결하고자 하는 파일이 폴더 상위에 있는 경우

같은 위치 또는 하위 경로일 경우 시작하는 명령어 : ./

상위 위치일 경우 시작하는 명령어 : ../ (한번 위 라는 뜻)
